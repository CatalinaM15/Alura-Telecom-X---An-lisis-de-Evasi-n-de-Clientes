# 📊 Telecom X — Análisis de Evasión de Clientes (Churn)

##  Descripción del Proyecto

Este proyecto analiza la evasión de clientes (*Churn*) en **Telecom X**, una empresa de telecomunicaciones que busca comprender por qué los clientes cancelan sus servicios.

El objetivo es identificar patrones y factores asociados con la cancelación de clientes mediante **análisis exploratorio de datos (EDA)** utilizando Python y herramientas de ciencia de datos.


---

#  Objetivos

- Analizar la distribución de clientes que cancelan el servicio.
- Identificar patrones asociados con la evasión.
- Explorar variables demográficas, contractuales y financieras.
- Generar insights que ayuden a reducir la tasa de cancelación.

---

# Tecnologías Utilizadas

- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Plotly**

---

# Fuente de Datos

Los datos fueron obtenidos desde una API pública en formato JSON:
https://github.com/ingridcristh/challenge2-data-science-LATAM


El dataset contiene información sobre:

- Datos demográficos de clientes
- Servicios contratados
- Tipo de contrato
- Método de pago
- Facturación mensual y total
- Estado de churn
